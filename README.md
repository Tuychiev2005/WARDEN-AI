# ⚡ Advanced Telegram Security Platform

Warden Security System — современная cyber-security платформа для мониторинга и анализа Telegram-сессий в реальном времени.

Проект сочетает:
- Telegram WebApp
- AI-анализ активности
- систему обнаружения угроз
- cyberpunk интерфейс
- защищённую backend архитектуру

---

# 🧠 Основные возможности

## 🔍 Threat Detection
Обнаружение подозрительной активности, scam-поведения и фишинговых атак.

## 🤖 Warden AI
Интеллектуальный модуль анализа Telegram-активности и аномального поведения.

## 🛡 Secure Backend
Защищённая архитектура на Python + FastAPI.

## ⚡ Real-Time Monitoring
Мониторинг Telegram-сессии в реальном времени.

## 🌐 Telegram WebApp
Современный WebApp интерфейс внутри Telegram.

## 🔐 Security Layer
- SHA-256 hashing
- bcrypt encryption
- session isolation
- encryption before database storage

# 🚀 Установка

## 1. Клонировать проект
git clone https://github.com/Tuychiev2005/Warden-Security-System.git
cd Warden-Security-System

## 2. Установить зависимости
pip install -r requirements.txt

## 3. Запустить сервер
python main.py
