Warden AI

Встроенный интеллектуальный модуль анализирует активность Telegram-сессии в реальном времени и автоматически выявляет угрозы.

🎣
Обнаружение фишинга
AI анализирует ссылки и тексты, выявляя поддельные сайты, кражу данных, мошеннические схемы.
🚫
Обнаружение скама
Распознаёт схемы мошенничества: фейковые инвестиции, «подарки», просьбы перевести деньги.
💬
Фильтрация спама и флуда
Анализируется частота сообщений и паттерны поведения. Массовые рассылки выявляются автоматически.
📊
Оценка уровня риска
Каждому событию присваивается: HIGH / MEDIUM / LOW для быстрой приоритизации.
⚡
Реакция в реальном времени
Анализ происходит в момент получения сообщения. Уведомление приходит спустя секунды.
🔐 Принципы защиты данных
Все данные сессии обрабатываются через защищённый backend и никогда не хранятся в открытом виде.
🔑
Hashing (SHA-256 / bcrypt)
Данные хешируются перед записью. Даже при утечке БД — исходные значения не восстановить.
🧂
Salt-защита
Каждый хеш уникален благодаря случайной соли. Исключает атаки по радужным таблицам.
🔒
Шифрование перед записью
Данные сессии шифруются до попадания в базу данных. Без ключа прочитать невозможно.
🌐
Защищённый backend
Вся обработка на стороне сервера. WebApp работает через HTTPS.
⏱
Rate limiting
Ограничение запросов с одного IP защищает от атак перебором.
⚠️ Внимательно прочитайте правила использования Warden перед началом работы.
📋 Правила использования
• Используйте только свои Telegram-аккаунты.
• Запрещено использовать систему для слежки за другими.
• Запрещено подключать чужие аккаунты без разрешения.
• Данные сессии нельзя передавать третьим лицам.
• Нарушение правил может привести к блокировке аккаунта Telegram.
🚫 Технические ограничения
• Одновременно поддерживается одна активная сессия на пользователя.
• История алертов хранится 30 дней, затем удаляется.
• Warden анализирует только входящие сообщения.
• Система не предназначена для автоматических действий с аккаунтом.
⚖️ Ответственность
Warden — инструмент защиты и мониторинга. Ответственность за способ использования лежит на пользователе.
